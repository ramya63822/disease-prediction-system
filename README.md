# 🏥 Disease Prediction using Machine Learning

An intelligent **AI-based symptom checker** that predicts possible diseases based on user-input symptoms and provides clinical insights such as risk levels, top conditions, and suggested lab tests.

---

## 🚀 Project Overview

This project is a **Clinical Decision Support System** designed to assist in early-stage disease analysis. Users can input symptoms, and the system analyzes them to predict the most probable disease using a rule-based logic engine.

The system also includes advanced features like emergency detection, contradiction analysis, and psychological pattern recognition.

---

## 🧠 Features

- ✅ Symptom-based disease prediction  
- 📊 Confidence score & Top 3 disease suggestions  
- 🚨 Emergency symptom detection  
- ⚠️ Contradiction analysis (e.g., weight loss vs weight gain)  
- 🧠 Stress-related pattern detection  
- 🧪 Suggested lab tests  
- 💡 Risk level indication (Low / Moderate / High)  
- ⚡ Real-time analysis (no backend required)

---

## ⚙️ Tech Stack

- **Frontend:** HTML, Tailwind CSS  
- **Logic Engine:** JavaScript  
- **Machine Learning (Planned):** Python (Random Forest, Scikit-learn)

---

## 🏗️ How It Works

1. User inputs symptoms  
2. Symptoms are normalized and processed  
3. Matching algorithm compares symptoms with disease database  
4. Scores are calculated based on matches  
5. System outputs:
   - Most probable disease  
   - Confidence score  
   - Risk level  
   - Top 3 predictions  
   - Suggested lab tests
